Responsive Nature Gallery

Explore these beautiful nature scenes. The layout adapts to your screen size for optimal viewing.

Responsive Design Features

  • Mobile-first design with single column layout on small screens
  • Tablet view (772px and above): Two columns with images at 45% width and max 30vh height
  • Large screen view (998px and above): Three columns with images at 30% width
  • All images include descriptive alternative text for screen readers
  • Keyboard navigation with Tab, Arrow keys, and Escape

Created as a responsive and accessible web project
